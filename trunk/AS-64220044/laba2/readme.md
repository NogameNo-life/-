## Вариант 13 - Аптека ##

```mermaid
flowchart TB
node1[Руководитель аптеки]
node2[Старший аптекарь]
```

|№ п/п|Наименование организационной единицы|Расшифровка|

|2|Старший аптекарь|Старший аптекарь|
|3|Фармацевт|Фармацевт|
